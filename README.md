# 🌅 Daily Reflection — Inspirational Quote Generator

A minimalist, contemporary web application that fetches random inspirational quotes to start your day with a fresh perspective. 

Unlike standard AI-themed developer templates that rely on neon blues, deep purples, or heavy dark-mode gradients, this project features a hand-tailored **warm, earthy contemporary palette** (terracotta, cream, charcoal, and soft sand accents) with sophisticated editorial typography.

## ✨ Features

- **Working API Integration:** Powered by the reliable `DummyJSON` API to instantly pull high-quality quotes on demand.
- **Modern Async/Await Architecture:** Clean, decoupled JavaScript using modern asynchronous patterns and robust error handling.
- **Thoughtful UI/UX:**
  - Complete viewport centering layout.
  - Interactive scale animations and micro-interactions on buttons.
  - Safe state management (buttons disable gracefully during network fetches to prevent duplicate requests)[cite: 4].
  - Clean typographic hierarchy using `DM Serif Display` for headers and `Inter` for interface elements[cite: 4].
- **Zero Framework Dependency:** Built purely using semantic HTML5, modern CSS3 variables, and Vanilla JavaScript[cite: 4].
